Foldwise 1.0.1

  • Correzione del loop di automazione: Risolto un problema per cui lo spostamento o la copia di file in una cartella di destinazione all'interno della cartella monitorata poteva attivare ripetutamente la stessa regola, causando la ripetizione del suffisso "2" nei nomi dei file.
  • Miglioramenti alla localizzazione: Migliorata la localizzazione in inglese e francese nella schermata dei risultati dell'analisi AI.
Aggiornamento precedente

Foldwise 1.0

  • Monitoraggio in tempo reale: Le cartelle scelte vengono osservate continuamente con FSEvents. Le regole si attivano nel momento esatto in cui arriva un file.
  • Motore di regole visuale: Crea automazioni con condizioni su tipo, nome, estensione, dimensione, data e contenuto testo PDF. Combina tutto con logica AND/OR.
  • Anteprima sicura: Prima di applicare una regola puoi vedere esattamente cosa succederebbe — file per file, azione per azione.
  • Log completo e Undo: Ogni azione viene registrata con data e percorso. Le azioni supportate sono annullabili direttamente dal log.
  • Regole schedulate: Esegui pulizie automatiche su base giornaliera, settimanale o a orario personalizzato.
  • AI locale privata: Analizza una cartella e suggerisce regole pronte all'uso, tutto sul tuo Mac senza nessun upload. (macOS 26+)
