Foldwise 1.0

  • Real-Time Folder Monitoring: Watched folders are tracked continuously using FSEvents. Rules fire the instant a new file appears — no delay, no polling.
  • Visual Rule Engine: Build automations using conditions on file type, name, extension, size, date, and readable text inside PDFs. Combine them with AND/OR logic.
  • Safe Preview Mode: Before enabling any rule, see exactly what would happen — file by file, action by action. No surprises.
  • Full Activity Log & Undo: Every action is recorded with a timestamp and path. Supported actions can be undone directly from the log.
  • Scheduled Rules: Run automatic cleanups on a daily, weekly, or custom schedule — perfect for batch archiving and periodic maintenance.
  • Private On-Device AI: Analyzes a folder and suggests ready-to-use rules based on what it finds. Everything runs on your Mac, nothing is uploaded. (macOS 26+)
