# Foldwise

**Foldwise** is a premium macOS app that keeps your files organized automatically.

It watches the folders you choose, understands the rules you define, and moves, renames, tags, or cleans up files for you. Built for people who want a calmer Desktop, a cleaner Downloads folder, and a file workflow that does not collapse under everyday clutter.

## A Smarter Way To Organize Your Mac

Foldwise turns file organization into a quiet background system. Instead of sorting files by hand, you create precise automations once and let the app keep your workspace in order.

Use Foldwise for:

- Downloads that sort themselves.
- Screenshots that move out of the way.
- PDFs and invoices routed into the right folders.
- Work files separated by name, type, size, date, or content.
- Daily cleanup routines for folders that collect too much noise.
- Safer automation with previews, logs, and undo support.

## Highlights

### Real-Time Folder Monitoring

Choose the folders that matter and Foldwise keeps an eye on them. When a new file appears, the app checks your active rules and performs the matching action automatically.

You stay in control: pause one folder, pause everything, or run rules manually whenever you want.

### Powerful Rules

Foldwise rules are flexible enough for real workflows but simple enough to manage visually.

Create automations based on:

- File type.
- Extension.
- File name.
- Readable text content.
- File size.
- File age.
- Creation date.

Combine conditions with "all" or "any" logic, set priorities, and choose whether a rule runs when files arrive or on a daily schedule.

### Professional File Actions

Foldwise can automatically:

- Move files.
- Copy files.
- Rename files.
- Send files to the Trash.
- Apply Finder tags.
- Run Shell or AppleScript actions.

When needed, Foldwise handles naming conflicts carefully to avoid unwanted overwrites.

### Preview Before You Run

Automation should feel safe. Foldwise can preview what would happen before applying rules to existing files, showing which files match, which rules would run, and what actions are planned.

### Activity Log And Undo

Every monitored folder includes a clear activity log. You can review completed actions, previews, errors, and monitoring changes.

For supported actions, Foldwise can also undo what happened, such as moving a file back, removing an applied tag, or deleting a created copy.

### Local Intelligence

Foldwise can analyze a folder and suggest useful rules based on what it finds. It can detect recurring patterns such as screenshots, invoices, duplicates, old files, file categories, and messy folders that need structure.

Suggestions are designed to be actionable: Foldwise helps you turn insight into an automation you can actually use.

### Menu Bar Control

Foldwise is always close by from the macOS menu bar. Open the app, check folder status, pause automation, or run rules without breaking your flow.

### Backup And Restore

Export and restore your automation setup, including folders, rules, conditions, and actions. Keep a backup of your workflow or move your configuration to another Mac with less friction.

## Privacy First

Foldwise is designed around local control.

- Your files stay on your Mac.
- Rules are stored locally.
- Folder access follows macOS security permissions.
- Local analysis does not require uploading your documents.
- Destructive cleanup uses the Trash instead of permanent deletion.

Internet access may be used for product features such as licensing and app updates.

## Foldwise Free

Foldwise is distributed as a freemium app. The free version gives you the essentials to start automating your file organization immediately.

Free includes:

- 1 monitored folder.
- Up to 3 rules per folder.
- Essential actions such as move and trash.
